glass morphism

What is glassmorphism?

Is a term used to describe UI design that emphasises light or dark objects, placed on top of colourful backgrounds. A background-blur is placed on the objects which allows the background to shine through â€“ giving it the impression of frosted glass.

How does it work?

The component should be semi-transparent and have some blur, both of which are true characteristics of glass. The CSS properties for this are background-color with rgba (the last parameter is transparency) and backdrop-filter with blur function.

CSS example:

.glass{
----
border-radius: 10px;
----
background-color:
----
rgba(255,220,240,0.3);
----
backdrop-filter: blur(8px);
}

fire effect
(hover)

Shadows and SVG turbulence

For this efect I made three filters with the SVG and deTurbulance HTML tags, and aplied it to the title. Then I added a bunch of shadows with colors ranging from blue, yellow to red, to represent the colors of the flame. A keyframe animation is applied to the title on hover of this card, so it switches between the three filters that are slighly different.

HTML:

<svg> <filter id="fire1"> <feTurbulence id ="turbulence" baseFrequency="0.1 0.1" numOctaves="5" seed="3"> </feTurbulence> <feDisplacementMap in="SourceGraphic" scale="6"></feDisplacementMap> </filter> <filter id="fire2"> <feTurbulence id ="turbulence" baseFrequency="0.09 0.12" numOctaves="5" seed="3"> </feTurbulence> <feDisplacementMap in="SourceGraphic" scale="6"></feDisplacementMap> </filter> <filter id="fire3"> <feTurbulence id ="turbulence" baseFrequency="0.12 0.25" numOctaves="5" seed="3"> </feTurbulence> <feDisplacementMap in="SourceGraphic" scale="6"></feDisplacementMap> </filter> </svg>

CSS:

----

----

----

----

----
0px -5px 10px #F00,
----

----
0px -2px 15px rgb(255, 217, 0),
----
0px 0px 15px rgb(255, 217, 0),
----
0px 0px 20px yellow,
----
0px 2px 10px #00F,
----
0px 2px 6px #00F;
----

----

----
