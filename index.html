glass morphism

What is glassmorphism?

Is a term used to describe UI design that emphasises light or dark objects, placed on top of colourful backgrounds. A background-blur is placed on the objects which allows the background to shine through â€“ giving it the impression of frosted glass.

How does it work?

The component should be semi-transparent and have some blur, both of which are true characteristics of glass. The CSS properties for this are background-color with rgba (the last parameter is transparency) and backdrop-filter with blur function.

CSS example:

.glass{
--
border-radius: 10px;
--
background-color:
--
rgba(255,220,240,0.3);
--
backdrop-filter: blur(8px);
}
